 Data Transfer Rate
        
 it is a term that describes how many data can be 
        
 transfered from source to destination device
        
 for example, from HDD to USB
        

        
 We also should understand measurments used for DTR
        
 and they are:
        
  1) Bits  per second bps (Mega/Giga bps or Mbps/Gbps)
        
  2) Bytes per second Bps (Mega/Giga Bps or MBps/GBps)
        
 
        
 If we are talking about: how to measure HDD DTR
        
 or what is hard disk drive data transfer rate,
        
 it is: sectors in each track * amount of heads
        
 DTR usually is used to measure the DTT which is 
        
 data transfer time or otherwise - access time

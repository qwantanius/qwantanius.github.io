HDD - hard disk drive
electro-mechanical (data can be stored and read
within mechanical HDD heads (disk read-n-write heads)
movements coordinated by HDD controller)
data storage device (data storage device
is an unit responsible for storing information)

TRACK
Track its simply a circle made BY ONE disk read and
write head.

SECTOR
Part of the track. Why we need it?
Sectors are needed because usually
sectore provides more room to store
data than needed amount
SECTOR is a sequence of BITS (e.g.: 010101010)
Another formal explanation: information on tracks is recorded as
continious
ALL SECTORS on the disc should contain the same amount of BIT's
Popular capacities are from 512 bytes to a few KB's

INTER-SECTOR GAPS
Check out brief guide ->
Gap between sectores with no data stored inside them
used for SECTOR's separation

CYLINDER
Set of tracks. In HDD there usually 4 r-w heads,
whenever they spin they create a 4 tracks.
these 4 tracks creates a cylinder.
Whenever position of those heads is changing - different
cylinders starts being accessable

ZBR
Zoned bit recording
method of storing information used by disks to optimize
data capacity within storing storing more data on outer
than inner tracks
the idea of zonning is that different zones are used for
for different data
More about ZBR
